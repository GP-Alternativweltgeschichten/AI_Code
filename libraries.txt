Im Terminal:

- Environment erstellen
conda create -n OlpeAI

- Environment aktivieren
conda activate OlpeAI

- Bibliotheken installieren
pip install fastapi uvicorn pydantic diffusers pillow torch

- Bibliotheken für GPU Verwendung
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu124

- Bibliothek für Test-Client
pip install requests

- Server starten
python ./Server/inpaint_REST.py

- Optional: Client starten (Nur zum Testen; Client hat ältere Version)
python ./Client/rest_client.py
